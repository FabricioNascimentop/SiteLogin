Flask backend

  • #interface desenvolvida com HTML5 e CSS responsivo
  • tecnologias e validações aplicadas no backend:
    • Flask-WTF:
      • validação de formulários de login, cadastro e recuperação de senha
      • proteção automática contra CSRF e mensagens de erro dinâmicas
    • JWT (JSON Web Token):
      • autenticação baseada em tokens seguros e reutilizáveis
      • controle de sessões sem armazenar dados no servidor
      • rotas protegidas que exigem autorização do token
    • SQLAlchemy + Hash:
      • uso do ORM para manipulação segura e prática do banco de dados
      • senhas criptografadas com algoritmo de hash (bcrypt ou werkzeug.security)
      • armazenamento seguro de credenciais e verificação eficiente durante o login
    • Flask Blueprint:
      • estrutura modularizada com separação de rotas por funcionalidade
      • facilidade para escalar e manter o projeto organizado
  • #um sistema seguro, modular e bem estruturado — resultado de estudo prático em Flask, voltado à criação de APIs profissionais com autenticação e manipulação de dados segura
